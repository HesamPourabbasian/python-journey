| Number Guessing Game *(optional warm-up)* | ⭐          | Loops, conditionals          |
| To-Do List                                | ⭐⭐         | Lists, functions             |
| Contact Book                              | ⭐⭐         | Dictionaries                 |
| Quiz Game                                 | ⭐⭐         | Functions, loops             |
| Movie Recommendation                      | ⭐⭐         | Sets, dictionaries           |
| Student Grade Manager                     | ⭐⭐⭐        | Most basic concepts together |
| Library Management                        | ⭐⭐⭐        | Nested data structures       |
| Inventory Manager                         | ⭐⭐⭐        | CRUD operations              |
| Banking System                            | ⭐⭐⭐        | Validation and exceptions    |
| Shopping Cart                             | ⭐⭐⭐⭐       | Nearly all concepts combined |

If your goal is to become job-ready, these projects build toward the kinds of CRUD (Create, Read, Update, Delete) logic you'll use in real applications. After completing them, you'll be well prepared to move on to file handling, object-oriented programming (OOP), modules/packages, and databases.
